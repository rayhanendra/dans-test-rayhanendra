# Github Jobs

## Description

Github Jobs is a simple web application that allows you to search for jobs on Github.

# Setup

1. Install dependecies
   ```bash
   yarn
   ```
2. Install husky
   ```bash
   yarn husky install
   ```
3. Setup husky
   ```bash
   chmod ug+x .husky/*
   ```

# Running / Building

- Running development
  ```bash
  yarn dev
  ```
